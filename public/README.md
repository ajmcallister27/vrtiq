This folder is served as `static` when the app is built by Vite. It is also where PWA icons and other assets are expected.

**Required files for PWA:**

- `favicon.ico` – the browser favicon
- `pwa-192x192.png` – 192x192 icon used in the manifest
- `pwa-512x512.png` – 512x512 icon used in the manifest
- `apple-touch-icon.png` – optional (can be a copy of the 192x192 icon)

Replace these placeholder images with your own application icons before building.
